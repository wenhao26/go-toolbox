# 🐿️ 统计松鼠 (Squirrel-Stat)

**统计松鼠** 是一款基于 Go 语言开发的高性能文件系统扫描与统计工具。它专为**超大规模目录**设计，能够利用多核并发能力，在极短时间内完成对百万级文件的数量与大小统计。

---

## 🌟 核心特性

* **极速并发**：摒弃传统的单线程 `filepath.Walk`，采用基于 `Channel` 的动态协程池架构，充分压榨 SSD 磁盘 I/O 性能。
* **海量支持**：支持 TB 级别目录统计，通过原子操作（Atomic）避免锁竞争，处理百万级文件不掉速。
* **智能容错**：自动跳过无权限目录、系统受保护文件及损坏的符号链接，确保统计过程不中断。
* **低内存占用**：采用任务池模式替代深度递归，严格控制内存开销，防止在深层目录中出现 OOM。
* **实时反馈**：毫秒级动态输出当前扫描路径，任务结束时精确汇总总耗时、文件总数及总大小。

---

## 🏗️ 架构设计

项目采用 **生产者-消费者 (Producer-Consumer)** 模型，通过 `Pipeline` 流水线处理数据：

1.  **任务分发 (Dispatcher)**：将根目录切分为多个子目录任务。
2.  **协程池 (Worker Pool)**：固定数量的轻量级 Goroutine 并发读取磁盘 `Dirent`。
3.  **原子累加器 (Atomic Counter)**：利用 CPU 硬件指令级原子加法，实现零锁竞争的统计。
4.  **格式化输出 (Formatter)**：将字节数转换为人类易读的 KB/MB/GB/TB 格式。
